## Basic Usage

1. データ定義を記した json ファイルを作成する

作成例: `hacker_news.json`

```json
{
  "title:`title text{}`": "string",
  "article[]:`tr.athing`": {
    "title:`td.title a text{}`": "string",
    "link:`td.title a attr{href}`": "string"
  }
}
```

```bash
$ web2json -j hacker_news.json https://news.ycombinator.com/
```
